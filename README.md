## About The Project

SpawnPoint is a multiplayer first-person shooter built from the ground up in **Unreal Engine 5** using **C++**. The project explores the full breadth of what it takes to build a competitive FPS experience - from responsive player movement and weapon mechanics to the networking systems that keep everything in sync across clients.

<p style="text-align: end">(<a href="#readme-top">back to top</a>)</p>

## Features

- **Multiplayer networking** - built on Unreal Engine's replication system, handling player state, weapon fire, and hit detection across the network
- **First-person gameplay** - fluid movement, aiming, and shooting mechanics designed for competitive play
- **Weapon system** - modular weapon architecture supporting multiple weapon types with accurate replication
- **Player state & scoring** - kill/death tracking, match state management, and session handling with an extended scoring system that rewards multi-kills (double, triple, rampage), kill streaks, headshots, and other combat achievements
- **Game modes** - structured match flow with lobby, in-progress, and end-of-match states
